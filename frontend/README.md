Frontend minimal Next.js app containing the administrator dashboard page.

To run locally:

1. Open a terminal and install dependencies inside `frontend`:

```powershell
cd 'c:\Users\eduar\Documents\prog2\projeto rissi - vs1\frontend'
npm install
npm run dev
```

2. Open `http://localhost:3000` in your browser.

Notes:
- The project includes simple UI stubs at `components/ui/*` so the page renders without additional libraries.
- If you prefer to use Tailwind/CSS frameworks, replace `styles/globals.css` and install the required packages.
